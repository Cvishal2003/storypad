# 2.7.0

What's New?

- New onboarding screen with calming animations
- Fixed draft saving issues – Resolved glitches when saving drafts.
- Enhanced security – Integrated secure storage for sensitive data.
- Better content display – Improved body trimming for previews on the homepage.
- Improved editor menu – Enhanced context menu for a smoother editing experience.
- Refined color scheme – Switched to monochrome colors for better dark/light mode consistency.
- New filter button – Easily filter stories with the new button.
- Share your stories – Added a "Share App" button for easy sharing.
- Multi-edit support – Now you can edit multiple stories on the search & tag pages.
