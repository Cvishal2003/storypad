# Marketing Guide

## Brand Positioning

**StoryPad** = Minimalist diary app with powerful features

### Key Messages

1. **Privacy-First:** "Your data stays yours. No ads, no tracking, no subscriptions."
2. **Simplicity:** "Timeline journaling. No folders, no tabs. Just your life, naturally."
3. **Customization:** "Minimalist design, maximum control. 1300+ fonts, rich formatting, your way."
4. **Open Source:** "Transparent, trustworthy, community-driven."

## Target Audience

**Primary:** General users who write and value minimalism

**Characteristics:**

- Appreciate clean, simple interfaces
- Want powerful features without complexity
- Value privacy and data ownership
- Prefer one-time purchases over subscriptions

## Competitive Advantages

| Feature       | StoryPad                   | Competitors               |
| ------------- | -------------------------- | ------------------------- |
| Pricing       | Free + optional add-ons    | Subscription-based        |
| Privacy       | Local-first, optional sync | Cloud-required            |
| Design        | Minimalist timeline        | Folder/notebook structure |
| Customization | Extensive editing tools    | Limited formatting        |
| Open Source   | GPL v3.0                   | Closed source             |
