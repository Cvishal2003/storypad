# Migrate from Spooky to StoryPad

## Migration Steps

### Step 1: Export Data from Spooky

1. Open **Spooky**.
2. Navigate to **Backups** -> **Offline Backup** (icon button).
3. Click **Export** and choose a folder to save the backup file.

### Step 2: Import Data into StoryPad

1. Download StoryPad if haven't:

   [![Play Store](https://img.shields.io/badge/Google_Play-414141?style=for-the-badge&logo=google-play&logoColor=white)](https://play.google.com/store/apps/details?id=com.tc.writestory)

2. Open **StoryPad** app.
3. Navigate to **Backups** -> **Offline Backup** (icon button).
4. Click **Import**, select the backup file you exported from Spooky, and click **Restore**.
5. Wait for the process to complete. Once done, your data will be available in StoryPad.
